Instructions:

  1. Enter Task Details:
    In the right-hand box, you'll find the Task Manager. Fill in the "Task Name" and "Task Description" fields.
  2. Select Status:
    Default status is "In Progress".
  3. Add Task:
    Click the "Add" button to add the task to your list.
  4. View Task List:
    The added tasks will appear in the table below the input form.
  5. Task Management:
    The table will display the ID, Name, Description, and Status of each task. Future updates will include actions to edit or delete tasks.
  6. ID Field:
    The ID field is read only, and will be automatically populated.
